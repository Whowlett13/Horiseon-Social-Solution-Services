# Usage

Accessibility has included simplified requirements, Have been tasked with a small list of specific criteria for this project. These criteria are documented in the Acceptance Criteria section.


# Acceptance criteria
GIVEN a webpage meets accessibility standards
WHEN I view the source code
I find semantic HTML elements have been simplified and classes have been changed
WHEN I view the structure of the HTML elements
I find that the elements follow a logical structure independent of styling and positioning
WHEN I view the icon and image elements
I find accessible alt attributes have been properly assigned a class and link associated
WHEN I view the heading attributes
They fall in sequential order
WHEN I view the title element
I find the name of the company
